# Landing Page Setup

The landing page files are:

- `index.html`
- `landing.css`

## Replace These Links

Open `index.html` and replace:

```text
https://t.me/YOUR_BOT_USERNAME
```

with your Telegram bot link, for example:

```text
https://t.me/BusyMomHelperBot
```

Replace:

```text
https://YOUR_PAYMENT_LINK
```

with your Gumroad, Ko-fi, PayPal, Shopify, or other payment link.

## Suggested Free Checklist Link

```

```

## Where To Host It For Free

Simple options:

1. GitHub Pages
2. Cloudflare Pages
3. Netlify

If you already uploaded this project to GitHub, GitHub Pages is the fastest path.

## GitHub Pages Setup

1. Open your GitHub repository.
2. Go to `Settings`.
3. Click `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`.
6. Folder: `/root`.
7. Save.

GitHub will give you a public landing page URL after a minute or two.
